# 增删改

# INSERT

sqala支持使用derives Entity的实体对象直接生成插入语句：

// 省略实体类定义
val people: People = ???

val i = insert(people)

支持使用对象列表批量写入：

val peopleList: List[People] = ???

val i = insert(peopleList*)

使用incrementKey标记的字段会在生成SQL时跳过。

当然我们也可以使用DSL来构建INSERT语句，这样就可以更精细地指定插入字段：

val i1 = insert into tablePeople(p => (p.id, p.name)) values (1, "小黑")

val i2 = insert into tablePeople(p => (p.id, p.name)) values List((1, "小黑"), (2, "小白"))

# UPDATE

对于derives Entity且使用sqala提供的注解配置了主键的实体对象，可以直接生成按主键更新的SQL：

// 省略实体类定义
val people: People = ???

val u = update(people)

可以额外传入一个skipNone参数，则会则更新时跳过值为None的字段：

val u = update(people, skipNone = true)

我们也可以使用DSL来构建UPDATE语句：

val u = update table tablePeople set (p => p.name === "小黑") where (_.id === 1)

# DELETE

我们可以使用DSL来构建DELETE语句：

val d = delete from tablePeople where (_.id === 1)
